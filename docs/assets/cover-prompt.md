# Cover banner — AI generation prompt

Generate the banner, then save it as `docs/assets/cover.png` (the README points
there). Target a wide banner ratio (**~3:1**, e.g. 1500×500). Best tools:
**Ideogram** or **GPT-4o/DALL·E 3** (they render text reliably);
**Midjourney v6** for the richest art (add the title in post if its text is off).

---

## Primary prompt

> A wide, premium tech banner for an open-source project called **"GOD-TIER GO"**.
> Clean, modern, slightly cinematic. A luminous, minimalist Go gopher mascot
> rendered as a calm, masterful artisan — subtly haloed in soft golden light,
> standing beside elegant flowing lines of Go source code that dissolve into
> particles of light. Deep midnight-blue to teal gradient background (Go brand
> cyan `#00ADD8` accents), thin geometric circuit/constellation lines, a sense of
> craftsmanship and quiet authority — "the highest tier of code." Crisp bold
> title text **"GOD-TIER GO"** in a clean geometric sans-serif, with a small
> tagline **"citation-backed Go agent skills"** beneath it. Soft volumetric
> lighting, high detail, vector-poster aesthetic, lots of negative space, no
> clutter. 3:1 ultrawide banner. --ar 3:1

## Style / parameter notes

- **Palette:** Go cyan `#00ADD8`, deep navy `#0B1B2B`, teal, with **gold** halo
  accents for the "god-tier" theme. Keep it 3–4 colors, high contrast.
- **Mood:** premium, calm, authoritative — *not* flashy/gamer neon.
- **Composition:** mascot/visual on one side, title + tagline on the other, or
  centered title with the motif framing it. Keep a clear safe area for text.
- **Avoid:** cluttered backgrounds, lens flares everywhere, fake/garbled code
  that looks like noise, stocky 3D render clichés, watermarks.

## Variations to try

- **Minimalist:** just the bold "GOD-TIER GO" wordmark on a navy→cyan gradient
  with a single thin golden underline and a faint constellation of 12 dots (one
  per skill). Very clean, great for a small banner.
- **Blueprint:** the gopher and code drawn as a glowing technical blueprint
  (cyan lines on navy), conveying "engineering precision."
- **Marble/divine:** the gopher as a serene marble statue with a faint gold
  laurel — leans into "god-tier" without being literal.
